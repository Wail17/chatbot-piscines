## 💻 Test Local (si besoin)

### Méthode 1: Serveur Python Simple
```bash
cd /home/user/chatbot-piscines
python3 -m http.server 8080
```

Puis ouvre: **http://localhost:8080/chatbot-demo.html**

Double-clique sur `chatbot-demo.html` pour l'ouvrir dans ton navigateur.

- **URL API**: `https://web-production-5dde.up.railway.app/chat`
